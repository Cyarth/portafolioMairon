PROYECTOS

Inventario gimnasio Duoc UC

Proyecto enfocado a resolver una problemática que tenia el Duoc UC de la sede Maipu en la gestión de inventario de su gimnasio. La solución constaba de una página web realizada en PHP con HTML5, CSS3 y bootstrap con base de datos MySQL.

Aplicación móvil

Proyecto utilizado para mi portafolio de título en Duoc UC, está hecha en IONIC con Angular consumiendo los datos mediante una API REST.
