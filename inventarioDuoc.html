GIMNASIO DUOC

Proyecto realizado en 4to año de estudio en donde se solicito solucionar una problematica con la escuela de salud del Duoc UC de la sede Maipu respecto a la gestion del inventario de los materiales dentro del gimnasio. Para esto desarrolle un sistema web con HTML5, CSS, JS, BOOTSTRAP, PHP, MySQL como base de datos. Github para controlar las versiones del sistema y trello para administrar el control de tareas dentro del proyecto.

El sistema cuenta con un formulario de login y registro, ya que sin cuenta no se puede entrar a la pagina. Una vez registrado con las credenciales ingresadas el usuario puede comenzar a subir stock de materiales, editar stock de materiales e incluso eliminar materiales. Tambien puede registrar solicitudes de reservas de materiales.

El sistema tambien cuenta con un apartado de reportes donde se puede ver quien ha ingresado o modificado materiales o reservas.
